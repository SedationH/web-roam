## yarn

[-W 添加到工作区](https://classic.yarnpkg.com/en/docs/cli/add/#toc-yarn-add-ignore-workspace-root-check-w)

```zsh
// -D dev -W workspace root
yarn add jest -D -W

// 注意是子的package.json中的名字
yarn workspace v-button add xxx
```
