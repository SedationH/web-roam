## chmod

文件权限修改

默认创建的文件📃

```zsh
$ ls -l file1
-rw-r--r--  1 sedationh  staff  0 Oct 19 19:50 file1
# 解释一下上面10个字符-rw-r--r-- 第一个表示文件类型 - 这里表示文件的意思
# 后面三个为一组 分别代表 拥有者的权限 同组权限 其他用户权限
# r read w write x exit
# r-4 w-2 x-1   -> 相加的值代表相应权限的集合
```

有时候我们需要执行这个文件

```zsh
$ chmod 775 file1 ## 对于拥有者和同组 可读可写可执行 其他人不可写 可读可执行
```
